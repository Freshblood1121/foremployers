============================================================================================================================

The repository contains a CRUD application that allows job seekers to manage their job openings.

## Functionality

- Create new vacancies: Candidates can add information about their profile.
- Search for vacancies: the ability to search for vacancies by ID, as well as display all vacancies.
- Job Updates: Candidates can edit their job vacancy details.
- Deleting vacancies: the ability to delete vacancies that are no longer relevant.
